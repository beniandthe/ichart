# iChart Recording Script: SM-001 Funk Groove

Status: capture complete
Date: 2026-07-16
Brief: `docs/marketing/social-media/brief-sm-001-handwrite-clean-gig-charts.md`
Demo material: `docs/marketing/social-media/demo-material-packet-001.md`

## Goal

Capture a short vertical-first workflow showing a bandleader/gigging musician using `Funk Groove` as a structured, reusable chart:

> Write one chart. Make band-ready versions. Send the PDFs.

## Core Claims

Say:
- Handwrite clean gig charts.
- Make chart versions for Concert, Bb Horn, and Eb Horn.
- Export/send PDFs.
- Store and edit later.

Avoid:
- Horn parts.
- Full notation.
- Automatic arrangement.
- Automatic paper cleanup.
- Importing or recognizing copyrighted songs.

## Final Video Shape

Target length:
- 20-28 seconds.

Master format:
- 1080x1920 vertical.

Best structure:
1. The bandleader problem.
2. The chart-writing moment.
3. The chart-version/transposition moment.
4. The PDF/share moment.
5. Quiet beta CTA.
