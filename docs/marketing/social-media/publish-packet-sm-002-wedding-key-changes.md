## Core Message

Last-minute key changes are normal on wedding and corporate gigs. iChart lets a musician keep the original chart, duplicate it, transpose the copy, store the updated key in the project, and reuse it later.

Use:

- "Duplicate the chart."
- "Transpose without rewriting."
- "Keep the original chart."
- "Store the new key with the set."
- "Open, export, send, or edit later."

Avoid:

- "Automatically makes charts."
- "Cleans up messy paper charts."
- "Imports from iReal Pro."
- "Replaces Finale."
- "Generates horn parts."
- "Arranges the song."
