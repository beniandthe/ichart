## Audience And Goal

- Primary audience: working bandleaders and gigging musicians responsible for getting charts to multiple players.
- User problem: paper-speed chart writing is fast, but paper charts are hard to transpose, duplicate into player-specific chart versions, send as clean PDFs, store, and edit later.
- Product truth to prove: iChart lets musicians handwrite clean charts quickly, then treat them like structured charts: duplicate, transpose, export, store, and revise.
- Desired action: follow for beta updates or ask to test on iPad.
- Landing page or App Store path: `https://useichart.com` until a dedicated beta/App Store path exists.
- UTM or custom product page URL: `sm001_funk_groove`.
- Demo chart/project: `Funk Groove`.

## Creative

### Hook

- Hook option 1: Write one chart. Make band-ready versions. Send the PDFs.
- Hook option 2: Paper-speed chart writing, without paper-chart problems.
- Hook option 3: One chart for rehearsal. Versions for the players.
