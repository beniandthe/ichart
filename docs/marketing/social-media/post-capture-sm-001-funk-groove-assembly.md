## Editing Tool Options

Available now:

- Local AVFoundation export: best for repeatable rough cuts, trim experiments, burn-in overlays, motion, and quick platform variants.
- iMovie: installed locally; best for hand-tuned timeline work, transitions, music, and a more natural editor review loop.
- Canva: useful for cover frames, graphic title cards, templates, resizing, and brand-kit visuals. The current exposed Canva tools are not a full local timeline editor for these private raw files.

Not available in the current tool surface:

- A dedicated CapCut/Final Cut/Premiere-style video editing plugin.
- `ffmpeg` / `ffprobe` command-line tooling.
