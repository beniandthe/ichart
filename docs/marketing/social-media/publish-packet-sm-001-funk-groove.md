## Core Message

One chart for the band. Clean handwritten input at paper speed. Player-ready chart versions, transposition, PDF export, and storage without rewriting from scratch.

Use:

- "Handwrite clean gig charts."
- "Make chart versions for the players."
- "Transpose for Bb or Eb players."
- "Export PDFs to the band."
- "Keep the chart for future edits."

Avoid:

- "Clean up messy paper charts."
- "Turn rough charts into clean PDFs."
- "Horn parts."
- "Full arrangements."
- "Automatic chart cleanup."
- "Finale replacement."
- "iReal Pro import/export."
