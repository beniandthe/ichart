## Audience And Goal

- Primary audience: wedding/corporate/event gig musicians and bandleaders.
- User problem: the set has many songs, singers want different keys, and rewriting paper charts is slow once the gig prep starts moving.
- Product truth to prove: iChart supports fast clean chord-chart writing, transposition, PDF export/share, and storage for future edits.
- Desired action: follow for beta updates or ask to test on iPad.
- Landing page or App Store path: `https://useichart.com` until a dedicated beta/App Store path exists.
- UTM or custom product page URL: `sm002_wedding_key_changes`.
- Demo project/chart: `Wedding Set - Key Changes` / `First Dance - C` -> `First Dance - F`.

## Creative

- Hook option 1: POV: the singer moved three songs and the band still needs charts.
- Hook option 2: Quick chord charts for new keys, without rewriting the page.
