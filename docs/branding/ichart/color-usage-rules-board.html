iChart color usage rules

Role-based guidance for applying the palette across the app without losing the focused musician-workstation feel.

role-based color
Dark frames the instrument. Paper holds the music.

Ink carries information. iChart Blue means action. Light Blue belongs to the logo. Blue Soft means quiet selection.

Do not drift into this

  • Dark editor canvas by default
  • Light Blue as link/control color on Paper
  • Blue notation for brand consistency
  • Low-contrast staff lines as functional UI
  • Gold/brass logo accents

Feedback States

Selected / pending

Blue Soft fill with iChart Blue outline or icon.

Saved / accepted

Quiet green with label or check icon.

Warning / unsure

Warm amber only when the user needs caution.

Error / destructive

Red with text/icon and clear separation.
